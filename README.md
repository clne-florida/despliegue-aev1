# Instrucciones
Se trata de intentar adivinar una palabra que nos da el sistema.  
Para jugar hay que hacer lo siguiente:  
1. Escribir la letra que creamos que contiene la palabra  
2. Darle a Adivinar  
3. Si la letra existe, nos permitirá seguir introduciendo letras  
4. Si no, nos restará 1 vida  
5. Si hemos acertado, nos redirigirá a una pantalla de victoria, permitiéndonos volver a jugar  
6. Si no, veremos la pantalla de derrota, que también nos permite volver a jugar
